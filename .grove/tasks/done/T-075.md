## Description

The Copilot model list is hardcoded in `src/main/ipc/plan.ts:229-239` and is outdated. The current list includes deprecated model names (e.g., `claude-3.7-sonnet` instead of `claude-sonnet-4.6`) and is missing newer models.

**Issues found:**

1. Hardcoded `COPILOT_MODELS` array is stale - missing `claude-opus-4.6`, `claude-opus-4.6-fast`, etc.
2. Model names use old format (e.g., `claude-3.5-sonnet` vs `claude-sonnet-4.x`)
3. No way to dynamically fetch models - Copilot CLI doesn't expose a `models` subcommand (unlike opencode)

**Root cause:** The Copilot CLI doesn't have an equivalent to `opencode models` - models are routed by GitHub Copilot's backend, so the list must be maintained manually.
