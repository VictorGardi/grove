## Description

Add a modal that displays all available keyboard shortcuts so users can reference them without memorizing them.

### UI Requirements

- Add a small `?` button in the bottom-right corner of the app (next to the sidebar toggle or in the title bar area)
- Clicking the `?` button opens a modal showing all keyboard shortcuts
- Only Mac keyboard shortcuts are displayed (not platform-specific)

### Modal Behavior

- Show a table of all shortcuts with columns: Shortcut, Action
- Close on Escape key, clicking the backdrop, or clicking the `?` button again
- Be accessible via Tab/Shift+Tab navigation with proper ARIA attributes
