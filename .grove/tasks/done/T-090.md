## Description

The kanban board columns (in `src/renderer/src/components/Board/Column.module.css`) currently show vertical scrollbars when there are more cards than fit in the column. The `.cardList` class has `overflow-y: auto` which displays native scrollbars.

The fix is to hide the scrollbar while maintaining scroll functionality via mouse wheel/trackpad:

1. Add `scrollbar-width: none` to `.cardList` (hides scrollbar in Firefox)
2. Add `&::-webkit-scrollbar { display: none; }` to `.cardList` (hides scrollbar in Chrome/Safari)
