## Description

Add window transparency settings to Grove, allowing users to adjust app window opacity. Users can set a transparency level in Settings (0% = invisible, 100% = fully opaque). Implemented as content opacity only (native window controls remain solid).

**Approach:** Use Electron's `BrowserWindow.setOpacity()` with a slider in the Settings UI. Config stored globally in `AppConfig` (like theme). Opacity clamped to 0.1-1.0 to prevent invisible windows.

**Known limitations:**

- Windows: System " transparency effects" setting may override app-level opacity
- Linux: Requires compositor support; may not work in all environments
