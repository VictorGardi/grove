## Description

Make nav bar buttons in the sidebar less prominent by hiding less-used buttons behind a collapsible section.

### Layout
- **Always visible**: Task Board, Settings (2 buttons at top and bottom)
- **Collapsible section**: Files, Decisions, Terminal, Agents (4 buttons hidden by default)
- An expand/collapse toggle control to show/hide the hidden buttons

### Behavior
- Clicking the expand control reveals the hidden buttons with their labels
- Hovering over a hidden button shows its label (when collapsed)
- When collapsed, only icons are shown for the collapsible items
- The Agents badge (running session count) is hidden when Agents is collapsed
- Collapse state persists to localStorage (`grove:nav-expanded`)

### Default state
- Collapsed (show only Task Board + Settings)
