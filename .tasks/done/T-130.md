## Description

Add a task-centric view as an alternative to the existing Kanban board view. The task-centric view displays three columns:

- **Left (reuses existing sidebar)**: Task list showing ALL tasks from ALL workspaces, grouped by workspace name and status (Backlog, Doing, Review, Done). Clicking a task switches the active workspace to that task's workspace.
- **Center (Primary Surface)**: Agent terminal/chat interface that switches mode based on task phase (planning or execution)
- **Right (Structured Context)**: Always-visible task details including description and live DoD checkboxes (clicking a checkbox automatically updates the markdown and saves)

Users can toggle between board view and task view from the main navigation. Clicking "Start Execution" initiates the same workflow as dragging a task from Backlog to Doing.
