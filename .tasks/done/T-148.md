## Description

The vertical column dividers in the kanban board are implemented via `border-right` on `.column` elements (Column.module.css:6). These borders do not extend to the full container height in most workspaces.

**Root cause**: The `.column` elements rely on flexbox stretching but lack explicit height constraints. The `.board` has `flex: 1` but no explicit `min-height: 100%`, and columns lack `height: 100%`.

**Fix**: Ensure the full height chain is explicit:

1. Add `min-height: 100%` to `.board` (Board.module.css)
2. Add explicit `align-items: stretch` to `.columns` (Board.module.css)
3. Add `height: 100%` to `.column` (Column.module.css)

Note: `align-items: stretch` is the flexbox default, but making it explicit ensures it's not overridden by any intermediate CSS. The `height: 100%` on columns ensures the `border-right` extends to match the container height.
