## Description

Add `--autopilot` flag to Copilot CLI when running in execution mode (non-plan tasks).

**What is autopilot mode**: Allows Copilot CLI to work autonomously through multiple steps without waiting for user input after each step. Once the initial instruction is given, Copilot works until the task is complete, an error occurs, or the user presses Ctrl+C.

**Implementation**: In `src/main/agentRunner.ts`, modify the `buildArgs` function to add `--autopilot` flag alongside `--allow-all-tools` when `mode !== "plan"`.

**Current code** (line 86-90):

```typescript
} else {
  // Execute mode: allow all tools for unattended headless runs.
  // Copilot defaults to asking for approval; --allow-all-tools suppresses prompts.
  args.push("--allow-all-tools");
}
```

**Change to**:

```typescript
} else {
  // Autopilot mode: copilot works autonomously without waiting for user input.
  args.push("--autopilot");
  args.push("--allow-all-tools");
}
```
