## Description

When the task switcher is open (triggered by Cmd+K) and the user presses Escape, the open task in the task detail panel gets closed. The Escape key should instead close the task switcher first, preserving the currently open task.

## Definition of Done

- [x] Add check for task switcher open state in Escape handler in useKeyboardShortcuts.ts
- [x] When task switcher is open and Escape pressed, only close the task switcher (not the selected task)
- [x] When task switcher is closed and Escape pressed, continue existing behavior (close selected task or clear search)
